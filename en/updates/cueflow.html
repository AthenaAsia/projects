CueFlow — Updates

Building

A service that automatically turns performance data (MIDI) into clean, readable sheet-music PDFs.

🔗 app.cueflow.jp ↗

Latest update

2026-06-21

  • Greatly strengthened handling of performance data with corrupted time-signature information. It now automatically finds the odd spots, suggests fixes, and remembers the settings you chose for next time.
  • Piano scores are now easier to read (removed unnecessary whitespace, combined the left- and right-hand staves into a single system, and merged fragmented time signatures).

History

2026-06-21

Handling broken performance data · clearer scores

Messy performance data can now be automatically detected and fixed. Piano-score rendering was also significantly improved.

Early 2026-06

Major layout improvements for multi-part pieces

For multi-part pieces such as string quartets, fixed broken measures and content overflowing the page. PDF quality for the main pieces is now stable.
